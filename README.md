# 💳 Bank API - Spring Boot Project

A simple REST API for banking system built using Spring Boot.

---

## 🚀 Features

- Create Account
- Get All Accounts
- Get Account by Name
- Deposit Money
- Withdraw Money
- Global Exception Handling
- Clean Layered Architecture (Controller - Service - Model)

---

## 🧱 Tech Stack

- Java 21
- Spring Boot
- Spring Web
- Maven

---

## 📡 API Endpoints

### Create Account
POST /accounts


### Get All Accounts
GET /accounts


### Get Account
GET /accounts/{name}


### Deposit
POST /accounts/{name}/deposit?amount=500


### Withdraw
POST /accounts/{name}/withdraw?amount=200


---

## 🧠 Concepts Covered

- Dependency Injection
- IoC Container
- REST API Design
- Exception Handling (@RestControllerAdvice)
- Service Layer Architecture
